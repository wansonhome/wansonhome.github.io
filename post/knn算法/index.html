knn

knn概述

Knn(k-NearestNeighbor)是一个分类算法，在机器学习中属于相对基础的方法之一。实现的核心思想，用人话来解释：

物以类聚，样本周围谁最多，它就是谁

具体解释为，在一个样本空间中的前k的距离最近的邻居，通常属于同一个类别，并具有这个类别上的特性。核心是根据其周围样本来进行分类判断，适合于类域交叉重叠较多的样本。

流程

  1. 预处理数据
  2. 划分测试样本和训练样本，比如三七分
  3. 维护一个大小为k的距离优先级队列，取测试样本计算它与训练样本的欧式距离或曼哈顿距离。
  4. 得出排名前k的点
  5. 投票确定分类

优缺点

简单好理解，适合于类域交叉重叠较多的样本。

计算量大，投票过程慢

Iris

数据集介绍

Iris数据集是常用的分类实验数据集，由Fisher, 1936收集整理。Iris也称鸢尾花卉数据集，是一类多重变量分析的数据集。数据集包含150个数据样本，分为3类，每类50个数据，每个数据包含4个属性。可通过花萼长度，花萼宽度，花瓣长度，花瓣宽度4个属性预测鸢尾花卉属于（Setosa，Versicolour，Virginica）三个种类中的哪一类。

Seeds

数据集介绍

共210个观察值，7个输入变量，一个输出变量 区域，周长，压实度，籽粒长度，籽粒宽度，不对称系数，籽粒腹沟长度

import numpy as np
import pylab as plt
from sklearn.datasets import load_iris 
from sklearn.model_selection import train_test_split #引入train_test_split函数
from sklearn.neighbors import KNeighborsClassifier   #引入KNN分类器

采用sklearn方法中的knn分类器，使用train_test_split方法对数据集七三分，进行训练和测试。

#使用train_test_split（）函数将数据集分成用于训练的data和用于测试的data
data_train,data_test,target_train,target_test=train_test_split(data,target,test_size=0.3,random_state=0)

#调用knn分类器
knn=KNeighborsClassifier()
#训练knn分类器
knn.fit(data_train,target_train)#调用该对象的打分方法，计算出准确率
accurate_Knn=knn.score(data_test,target_test,sample_weight=None)

print('KNN预测值：',knn.predict(data_test))#预测值
print('真实值：',target_test)#真实值
print('KNN输出训练集的准确率为：',accurate_Knn)
